# 🤖 AI-Кротик: Telegram Bot с поддержкой ИИ

### 📂 Структура проекта
- `bot.js` — **Ядро**. Инициализация Telegraf, входная точка для всех сообщений.
- `modules/` — **Модули**:
  - `config.js` — Все константы: токены, ID админа, пути к файлам и список моделей.
  - `storage.js` — Логика чтения/записи JSON-файлов (БД бота).
  - `menus.js` — **UI/UX**. Описание всех кнопочных меню (Keyboard).
  - `handlers.js` — **Контроллер**. Логика переключения между меню и обработки кнопок.
  - `commands.js` — Регистрация команд (начинаются с `/`).
  - `broadcast.js` — Сервис массовой рассылки сообщений.
  - `ai.js` — Интеграция с API нейросетей.
- `data/` — **База данных**:
  - `settings.json` — Настройки пользователей (выбранная модель, режим).
  - `history.json` — История переписки для контекста ИИ.
  - `broadcasts.json` — Логи рассылок.

### 🛠 Логика работы
1. **Сообщения**: Все входящие попадают в `bot.js`.
2. **Фильтрация**: Сначала проверяется, не находится ли админ в режиме рассылки.
3. **Меню**: Если текст совпадает с кнопкой, `handlers.js` меняет экран.
4. **ИИ**: Если текст не команда и не кнопка, он уходит в `ai.js`.

### Эти библиотеки обеспечивают работу бота:
- telegraf: Основной фреймворк для работы с Telegram API.
- axios: Для отправки HTTP-запросов к API нейросетей (внутри ai.js).
- fs: (Встроенный модуль Node.js) Для работы с файловой системой (чтение/запись JSON).
